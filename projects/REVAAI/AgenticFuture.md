# 🤖 The Agentic Future: Beyond Chatbots

Dr. Manish Gupta's RAISE'25 address highlighted a fundamental shift in AI development: the transition from **Large Language Models (LLMs)** to **Large Multimodal Models (LMMs)** and ultimately to **Autonomous Agents**.

## 🔄 The Interaction Shift
- **Yesterday (Predictive)**: LLMs focused on predicting the next token in a text sequence.
- **Today (Interactive)**: Multimodal models (like Gemini) process video, audio, and text natively.
- **Tomorrow (Agentic)**: Agents that don't just "talk" but "perform" complex, multi-step tasks autonomously.

## 👁️ Project Astra: Real-time Awareness
**Project Astra** is a pioneer in universal agents.
- **Capabilities**: It can process video and audio streams in real-time.
- **Memory**: Remembers where you left your keys or what you were talking about minutes ago.
- **Low Latency**: Natively multimodal architecture allows for human-like response speeds (demonstrated using smart glasses and smartphones).

## 🚢 Project Mariner: Professional Autonomy
**Project Mariner** focuses on task-oriented agents.
- **Autonomy**: Unlike simple chatbots, Mariner can navigate complex workflows.
- **Example Use Cases**: 
    - Managing enterprise software interfaces.
    - Researching and synthesizing deep-web information.
    - Coordinating multi-party logistics.
- **Reasoning**: Combines high-level planning with low-level execution for reliable task completion.

## 🧩 Key Technologies Enabling Agents
1. **Native Multimodality**: No longer "bolting on" vision to a text head. Models like [[Gemini]] are trained on all modalities simultaneously.
2. **Context Windows**: Extremely long context windows (1.5M+ tokens) allow agents to "live" in a project for extended periods with full memory.
3. **Reasoning Loops**: Chain-of-thought and self-correction mechanisms that allow agents to plan before they act.
