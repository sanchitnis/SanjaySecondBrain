# Tag Taxonomy

This file defines the controlled vocabulary for tags in this wiki. Use this to ensure consistency and discoverability.

## Domains

- #ai: Artificial Intelligence, LLMs, agents, machine learning.
- #pkm: Personal Knowledge Management, Obsidian, productivity.
- #coding: Software development, languages, frameworks.
- #meta: Content about the wiki itself, instructions, setup.

## Technical Layers

- #concept: Abstract ideas and mental models.
- #entity: People, companies, tools, specific actors.
- #project: Initiatives with a clear goal and timeline.
- #reference: Sources, books, papers, articles.
- #synthesis: Combined reports and high-level summaries.

## Status Tags

- #todo: Action required on this page.
- #rough: Initial drafting phase.
- #vetted: Information has been cross-checked and verified.
- #archived: Information is historical or superseded.
